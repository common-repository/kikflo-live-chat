== Description ==

Kikflo Live Chat allows you to chat with visitors to your site via a customizable chat window.  You can customize the chat window to match your site by changing colors, fonts, and other details.

== Installation ==

1.  Go to www.kikflo.com and start an account.
2.  Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Do I need an account at kikflo.com? =

Yes, to use the chat, first go to kikflo.com and start an account.
